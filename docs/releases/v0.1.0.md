# ClipH By Henry Gossou — v0.1.0

Première version publique de ClipH, un gestionnaire moderne et persistant
de presse-papiers conçu pour Linux.

## Fonctionnalités principales

- Historique local des textes, contenus riches, images et fichiers
- Classification automatique des contenus copiés
- Recherche rapide dans l’historique
- Catalogue hors ligne d’emojis, kaomojis et symboles Unicode
- Raccourci global Super/Windows + H
- Raccourci de secours Ctrl + Super/Windows + H lorsque Super + H est occupé
- Détection automatique des conflits avec les raccourcis GNOME
- Démarrage automatique avec systemd utilisateur
- Conservation séparée des données pour chaque utilisateur
- Icône officielle et intégration au menu des applications
- Paquet Debian professionnel pour Ubuntu et Debian
- Page de manuel accessible avec `man cliph`
